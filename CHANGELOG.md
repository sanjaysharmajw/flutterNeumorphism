## 1.0.2

* Added `FlutterNeumorphism` widget — new simple API with optional width, height, and onTap
* Added `NeumorphismStyle` class — bundles all styling with auto-computed shadow colors
* Added `NeumorphismType` enum — `flat` (raised) and `pressed` (inset) styles
* Added `BoxShape.circle` support for circular neumorphic buttons and avatars
* Updated example app with 3 modern screens: Smart Home dashboard, Music Player, and Profile
* Updated README with full API documentation and parameter tables
* Upgraded Android Gradle Plugin to 8.6.0 and Kotlin to 2.1.0
* Upgraded Gradle wrapper to 8.7
* Original `FlutterNeumorphisms` widget kept unchanged for backward compatibility
